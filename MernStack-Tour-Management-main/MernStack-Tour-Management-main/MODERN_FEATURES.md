## 📋 **Installation & Setup**

### **Backend Setup:**
```bash
cd backend
npm install
npm start
```

### **Frontend Setup:**
```bash
cd frontend
npm install
npm start
```

### **Environment Variables:**
- `STRIPE_SECRET_KEY`: Stripe API key
- `STRIPE_PUBLISHABLE_KEY`: Stripe public key
- `TWILIO_ACCOUNT_SID`: Twilio account SID
- `TWILIO_AUTH_TOKEN`: Twilio auth token
